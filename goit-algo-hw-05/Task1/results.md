# Висновки щодо швидкості алгоритмів пошуку

Для порівняння було використано два текстові файли: `text1.txt` та `text2.txt`.
У кожному файлі перевірявся один підрядок, який справді існує в обох текстах,
та один вигаданий підрядок.

- Існуючий підрядок: `структури даних`
- Вигаданий підрядок: `вигаданий фрагмент, якого точно немає у статтях`
- Кількість запусків для одного заміру: `100`

## Висновки

Для `text1.txt` найкращий результат показав алгоритм Боєра-Мура. Він був
найшвидшим як для існуючого підрядка, так і для вигаданого підрядка.

Для `text2.txt` при пошуку існуючого підрядка результати алгоритмів Боєра-Мура та Кнута-Морріса-Пратта дуже близькі, оскільки підрядок знаходиться на початку тексту. При пошуку вигаданого підрядка алгоритм Боєра-Мура показав значно кращий результат.

У цілому найкращим серед трьох алгоритмів для цих двох текстів можна вважати
алгоритм **Боєра-Мура**, оскільки він показав найстабільніший і найшвидший результат
у більшості вимірювань. Алгоритми Кнута-Морріса-Пратта та Рабіна-Карпа також
коректно знаходять підрядки, але для заданих текстів і вибраних підрядків
працюють повільніше.
